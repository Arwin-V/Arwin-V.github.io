Engine Architecture

Engine modularity: Decoupling Hardware Abstraction (SFML) from the ECS Simulation layer.

Core Engineering Implementations

  1. Data-Oriented ECS Architecture: Implemented a cache-friendly Entity Component System (ECS) to manage concurrent entity processing. By decoupling simulation data from behavioral logic, the engine maximizes CPU cache locality, drastically reducing memory latency.
  2. Deterministic Fixed-Timestep Loop: Engineered a robust accumulator-based master loop. Physics and simulation logic are strictly decoupled from rendering framerates, ensuring deterministic behavior across variable hardware.
  3. Inversion of Control (IoC) Pipeline: Standardized engine lifecycle management where the `Application` delegates hardware abstraction to the `Engine` core, while the `Sandbox` client defines game-specific state, ensuring strict system isolation.
  4. Networked Listen-Server Model: Architected a UDP-based networking layer using binary serialization. Implemented state snapshots and RPC marshaling to maintain synchronization between client and server, masking latency through state interpolation.
  5. Dear ImGui Debug Overlay: Integrated an interface-driven debug layer. Implemented an `IDebuggable` interface for all engine subsystems, allowing the overlay to dynamically query system states at runtime without impacting production build performance.
  6. Resource Management & Build System: Managed build complexity through a modular CMake workspace, utilizing `FetchContent` for dependency resolution (SFML/ImGui). Enforced static linking to minimize deployment footprint and runtime dependency conflicts.

// ------------------------------------------------------------------------------------------------
// Engine.h - Interface Definition
// ------------------------------------------------------------------------------------------------
#pragma once
#include <SFML/Graphics.hpp>
#include <string>
#include <unordered_map>

namespace Flow {
    struct EngineConfig {
        uint32_t WindowWidth = 800;
        uint32_t WindowHeight = 600;
        std::string WindowName = "Flow Engine";
        bool bIsFullscreen = false;
    };

    class Engine {
    public:
        Engine() = default;
        ~Engine() = default;

        void InitializeEngine();
        void ShutdownEngine();

        // Const-correct accessors for system integrity
        const sf::RenderWindow& GetWindow() const noexcept { return Window; }
        bool IsRunning() const noexcept { return bIsEngineRunning; }

    private:
        sf::RenderWindow Window;
        bool bIsEngineRunning = false;
        EngineConfig Config;
        std::unordered_map<std::string, std::string> RawConfigData;

        void LoadConfigurationFile(const std::string& FilePath);
    };
}

// ------------------------------------------------------------------------------------------------
// Engine.cpp - Implementation
// ------------------------------------------------------------------------------------------------
#include "Engine.h"
#include <fstream>

namespace Flow {

    void Engine::LoadConfigurationFile(const std::string& FilePath) {
        std::ifstream File(FilePath);
        if (!File.is_open()) return;

        std::string Line;
        while (std::getline(File, Line)) {
            const size_t DelimiterPos = Line.find('=');
            if (DelimiterPos != std::string::npos) {
                RawConfigData[Line.substr(0, DelimiterPos)] = Line.substr(DelimiterPos + 1);
            }
        }
        
        // Defensive parsing with default fallbacks
        if (RawConfigData.count("WindowWidth")) Config.WindowWidth = std::stoul(RawConfigData.at("WindowWidth"));
        if (RawConfigData.count("WindowName")) Config.WindowName = RawConfigData.at("WindowName");
    }

    void Engine::InitializeEngine() {
        LoadConfigurationFile("Engine.ini");

        const sf::VideoMode Mode{ Config.WindowWidth, Config.WindowHeight };
        const sf::Style Style = Config.bIsFullscreen ? sf::Style::Fullscreen : sf::Style::Default;
        
        Window.create(Mode, Config.WindowName, Style);
        bIsEngineRunning = Window.isOpen();
    }

    void Engine::ShutdownEngine() {
        if (Window.isOpen()) Window.close();
        bIsEngineRunning = false;
    }
}
